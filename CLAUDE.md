# SecondMe 个人信息网站

## 项目概述
这是一个集成 SecondMe OAuth 登录和个人信息获取的网站项目。

## 功能模块
- ✅ OAuth 身份验证
- ✅ 个人信息获取和展示

## 技术栈
- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma (SQLite)
- SecondMe API

## 开发指南
1. 运行 `npm install` 安装依赖
2. 运行 `npx prisma db push` 初始化数据库
3. 运行 `npm run dev` 启动开发服务器

## 环境变量
项目使用 `.env.local` 文件存储敏感配置信息。
